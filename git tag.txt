git tag

git tag "V0.1*"

git -a V1.0 -m "This is a msg Version 1.0"

git show V1.0

git push origin V1.0



TO delete a tag:

git tag -d V1.0
git push origin :refs/tags/V1.0


TO get the commit HEAD (short version):

git rev-parse --short HEAD | clip

or 

git rev-parse HEAD | clip
